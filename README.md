# 基本操作


 - コンテナのbuild
```cmd
docker-compose build
```


- コンテナのrun
 ```cmd
docker-compose up -d
```


 - コンテナのstart
```cmd
docker-compose start

```


- docker環境に入る
```cmd
docker-compose exec myservice bash
```


 - docker環境からCMD
```bash
exit
```


- コンテナのstop
```cmd
docker-compose stop
```
